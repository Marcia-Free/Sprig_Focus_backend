# Sprig Focus backend

A cute gamified Task Manager built with Javascript and BabylonJS.


## Built With
* BabylonJS
* Javascript frontend
* React and Redux
* JWT Tokens
* SemanticUI with HTML/CSS
* PostgreSQL database


### Install Instructions
1. ```bundle install```
2. ```rails s```
3. Start frontend project [here](https://github.com/Marcia-Free/Sprig_Focus_frontend)
